JsonPath 表达式引用 JSON 结构，使用 $ 分配给外层对象的抽象名称。

| JsonPath 表达式     | 描述                        |
|------------------|---------------------------|
| $                | 根对象或元素                    |
| @                | 当前对象或元素                   |
| . or []          | 子元素操作符                    |
| ..               | 递归子元素                     |
| *                | 通配符，匹配所有对象或元素             |
| []               | 下标运算符                     |
| [,]              | 连接运算符，将多个结果拼接为数组返回，允许使用别名 |
| [start,end,step] | 数组切片                      |
| ?()              | 过滤器(脚本)表达式                |
| ()               | 脚本表达式                     |
